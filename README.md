### *Как запускать?

1. Убедитесь что у вас установлен `node`
2. Выполните команду `yarn`
3. Выполните команду `yarn dev`

### Линтинг

```yarn lint```

### Форматирование prettier

```yarn format```

### Production build

```yarn build```

и посмотреть, что получилось

`yarn preview`
